# PGA Stats Tracker

Full-stack web application for searching PGA Tour players and viewing their statistics and tournament results.

## Tech Stack

**Frontend:**
- React 18
- React Router
- Tailwind CSS
- Axios
- Vite

**Backend:**
- FastAPI (Python)
- SQLAlchemy ORM
- SQLite
- SportsData.io API

## Features

- Search PGA Tour players by name
- View player profiles with career stats
- Display recent tournament results
- Responsive design with Tailwind CSS

## What I Learned

- Building RESTful APIs with FastAPI
- React component architecture and routing
- Integrating external APIs with fallback data
